# Hello App

A simple web app that responds to user input.

## Features
- Clean, modern UI
- Type "hello" to get a response ("Bye!" on the bye-message branch)
- Handles unknown input gracefully

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/gigizee02/8004.git
   ```
2. Open `index.html` in your browser.

## Development
- Edit `index.html` to change the UI or logic.
- Use feature branches for changes and submit pull requests.
